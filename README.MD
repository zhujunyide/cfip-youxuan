# Cloudflare 优选 IP

**测试地**: 中国四川联通  
**更新频率**: 每半小时自动更新  
**本次更新**: 2026-05-08 08:59:17

## 订阅链接

- **高速优选**：`https://raw.githubusercontent.com/HandsomeMJZ/cfip/refs/heads/main/best_ips.txt`
- **所有可用**：`https://raw.githubusercontent.com/HandsomeMJZ/cfip/refs/heads/main/full_ips.txt`

-**区别**:  
**所有可用**是筛选出来每个地区延迟最低的前十个,不保证速度是最快的,适合二次自己测速.  
**高速优选**是筛选出来的里面已通过测速的,小白直接选这个.

## 说明
已有大量反馈大部分地区的**电信**和**联通**都可以直接使用此订阅  
如果你的网络环境不适用可以手动下载工具来自己优选  
如果不需要同步到github功能直接运行对应系统的start脚本即可，无需修改  
如果需要同步到github功能需要打开对应系统的push脚本自行修改token和仓库链接  

## 同款工具
**仓库链接**：`https://github.com/HandsomeMJZ/cfip-tools`  
有直接编译好的`windows版本`可供下载  

## 使用方法

1. 复制订阅链接。
2. 打开 EdgeTunnel 后台。
3. 创建优选订阅，选择“优选订阅模式 - 自定义”。
4. 在订阅接口 / API / URL 中粘贴订阅链接。
5. 验证可用性，选择“追加 API”。
6. 保存并重新订阅。
